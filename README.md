# 🛒 Amazon Clone – Frontend

This project is a **responsive clone of the Amazon homepage** built using **HTML and CSS**.  
The goal of this project was to replicate the layout and design of the Amazon website while improving frontend development skills.

The project was created while learning from **Apna College**, and special thanks to **Shraddha Khapra** for the guidance that helped in understanding the structure of real-world websites.

---

## 🎯 Project Objective

The main purpose of this project was to:

- Understand large website layouts
- Practice HTML and CSS positioning
- Build responsive UI components
- Replicate real-world e-commerce interface design

---

## 🛠️ Tech Stack

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout design  
- **Font Awesome** – Icons  

---

## ✨ Features

- Responsive Amazon-style navigation bar
- Location and delivery section
- Search bar with category selector
- Product category cards
- Hero section with promotional banner
- Multiple product sections
- Amazon-style footer with links
- Clean and interactive UI
